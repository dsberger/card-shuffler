Card Shuffler

by Dan Berger

This is a demo app to show drag-and-drop functionality using a combination of Ruby on Rails and Angular.js.

The code can be found here on Github.

Each list and card displays a 'front-end position' and a 'back-end position'.

The front-end position is calculated dynamically by Angular with reference to array indices and object names.

The back-end position is an attribute of the model in the Rails database.

When you move a list or card, you may notice a lag while the positions are out of sync.

The back-end postion is updated with a success callback after the AJAX request is done running.
